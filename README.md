# WorldHome
WorldHome Spring Project   

프로젝트 설명
장기간 자신이 소유한 집을 비워야할 상황일때 다른사람이 기간을 예약하고 돈을 지불하여 집을 사용할 수 있도록 만든 사이트   

프로젝트 목적   
상업용이 아니고 포트폴리오를 위해 만듦   
Security4를 이용한 로그인 및 권한 설정   
Tiles를 적용한 화면영역 분리   
Locale을 이용한 메시지 처리      

개발환경    
Window10 ( 배포서버는 AWS Centos7 )     
Oracle19C ( 배포서버는 Oracle11g )    
OpenJDK13    
Tomcat9    
SpringFramework    
Mybatis    
Maven
